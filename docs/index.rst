Welcome to uiucprescon.build
============================
Welcome to uiucprescon.build's documentation. uiucprescon.build is a pep517
builder for Python that helps integrate C or C++ dependencies or bindings.

This tool heavily relies on PyBind11 building extensions and Conan to manage
C++ dependencies.
